# 🦾 RS-Claw 使用指南

> 把电脑变成听话的 AI 助手——你说人话，它帮你干活。

---

## 什么是 RS-Claw？

RS-Claw 是一个 **AI 电脑助手**。你像跟朋友聊天一样告诉它你要做什么，
它就能真的帮你在电脑上操作：查文件、写代码、执行命令、上网查东西。

**你不需要会任何技术**，跟它说话就行。

而且它会**学习**——你每次用它做完一件事，它会自动总结经验变成技能，下次更快。
对话也会自动保存，下次打开继续聊。

## 第一步：拿到 RS-Claw

下载对应你系统的文件（约 6 MB），把它放到一个方便找的地方，比如**桌面**。

> ⚠️ 如果 Windows 弹出「Windows 已保护你的电脑」，点「**更多信息**」→「**仍要运行**」。

---

## 第二步：填写 API Key（只需要做一次）

RS-Claw 背后连的是 DeepSeek 大模型，你需要一个 Key 才能用。

### 2.1 获取 DeepSeek API Key

1. 打开浏览器，访问：**https://platform.deepseek.com**
2. 注册账号（用手机号或邮箱）
3. 登录后在左侧菜单找到「**API Keys**」
4. 点「**创建 API Key**」，复制那一长串以 `sk-` 开头的字符
5. 充值 10 块钱就够用很久（DeepSeek 非常便宜）

### 2.2 配置 RS-Claw

打开命令行，进入你放 RS-Claw 的目录，输入：

```
rs-claw.exe setup
```

然后按提示操作：

```
Provider [deepseek]:    ← 直接按回车
API Key:               ← 粘贴你的 sk-xxx key
Model [deepseek-chat]:  ← 直接按回车
```

看到「Configuration saved」就完成了！以后不用再配置。

## 第三步：开始使用

### 方式一：聊天模式（推荐）

双击 `rs-claw.exe` 或在命令行输入 `rs-claw`，出现 `>` 提示符后直接打字：

```
> 帮我看看桌面有什么文件
> 写一首关于春天的五言绝句
> /quit                          ← 退出
```

退出时会自动保存对话，下次打开继续聊。

### 方式二：一句话命令

```
rs-claw.exe chat -m "帮我统计一下C盘还剩多少空间"
```

## 常用命令

| 命令 | 作用 |
|:---|:---|

## 常见问题

**Q: 提示「API Key not configured」怎么办？**
A: 重新做一遍第二步。

**Q: 提示「DeepSeek API error」怎么办？**
A: 检查 API Key 是否正确，去 DeepSeek 后台看看余额。

**Q: 会不会把我电脑搞坏？**
A: RS-Claw 只能读文件和执行你让它做的命令。如果担心，可以说「先告诉我你要做什么，别直接执行」。

**Q: 聊天记录会丢吗？**
A: 不会。每次对话自动保存，下次启动自动恢复。用 `/sessions` 可以看历史。

**Q: 需要联网吗？**
A: 需要。它要连 DeepSeek 的服务器才能思考。

**Q: 怎么推荐给朋友？**
A: 把文件发给朋友，让他们也填自己的 Key 就行。
