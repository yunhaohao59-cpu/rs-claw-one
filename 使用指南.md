# 🦾 RS-Claw 使用指南

> 把电脑变成听话的 AI 助手——你说人话，它帮你干活。

---

## 什么是 RS-Claw？

RS-Claw 是一个 **AI 电脑助手**。你像跟朋友聊天一样告诉它你要做什么，
它就能真的帮你在电脑上操作：查文件、写代码、执行命令、上网查东西。

**你不需要会任何技术**，跟它说话就行。

---

## 第一步：拿到 RS-Claw

同学会发给你一个文件叫 `rs-claw.exe`（约 4MB）。

把它放到你电脑上一个好找的地方，比如 **桌面** 或 **D盘根目录**。

> ⚠️ 如果 Windows 弹出「Windows 已保护你的电脑」，点「**更多信息**」→「**仍要运行**」。

---

## 第二步：填写 API Key（只需要做一次）

RS-Claw 背后连的是 DeepSeek 大模型，你需要一个 Key 才能用。

### 2.1 获取 DeepSeek API Key

1. 打开浏览器，访问：**https://platform.deepseek.com**
2. 注册账号（用手机号或邮箱）
3. 登录后在左侧菜单找到「**API Keys**」
4. 点「**创建 API Key**」，复制那一长串以 `sk-` 开头的字符
5. 充值 10 块钱就够用很久（DeepSeek 非常便宜）

### 2.2 配置 RS-Claw

打开 RS-Claw：
- **双击** `rs-claw.exe`

它会提示你没有配置，教你配置。你也可以在命令行里输入：
```
rs-claw.exe setup
```

然后按提示操作：
```
Provider [deepseek]:    ← 直接按回车
API Key:               ← 粘贴你的 sk-xxx key
Model [deepseek-chat]:  ← 直接按回车（这里可以修改为其他模型，比如 deepseek-v4-pro 等，默认是 deepseek-chat）
```

看到「Configuration saved」就完成了！以后不用再配置。

---

## 第三步：开始使用

### 方式一：聊天模式（推荐）

**双击** `rs-claw.exe`，出现 `>` 提示符后直接打字聊天：

```
> 帮我看看桌面有什么文件
> 写一首关于春天的五言绝句
> 帮我查一下今天天气
> /quit                          ← 退出
```

### 方式二：一句话命令

在命令行里直接：
```
rs-claw.exe chat -m "帮我统计一下C盘还剩多少空间"
```

## 它能做什么？

下面是 RS-Claw 会用的 **7 大技能**，它会自动判断该用哪个：

| 技能 | 能干嘛 | 这样跟它说 |
|:---|:---|:---|
| 📁 读文件 | 查看文件内容 | 「帮我看一下 notes.txt 写了什么」 |
| ✏️ 写文件 | 创建或修改文件 | 「帮我建一个 hello.txt 里面写你好世界」 |
| 📂 列目录 | 看看文件夹里有什么 | 「桌面有哪些文件」 |
| 🔍 检查文件 | 确认文件有没有 | 「帮我看看D盘有没有那个报告」 |
| 💻 执行命令 | 运行终端命令 | 「帮我看看电脑IP地址是多少」 |
| 🌐 上网查 | 发 HTTP 请求 | 「帮我看看百度能不能访问」 |
| 📫 发请求 | POST JSON 数据 | 「帮我给这个接口发个测试请求」 |

## 常见问题

**Q: 提示「API Key not configured」怎么办？**
A: 重新做一遍第二步，确保 key 填对了。

**Q: 提示「DeepSeek API error」怎么办？**
A: 检查 API Key 是否正确，去 DeepSeek 后台看看余额。

**Q: 它说「抱歉我不行」？**
A: 换个说法试试。比如不说「杀毒」，说「帮我看看电脑有没有异常进程」。

**Q: 会不会把我电脑搞坏？**
A: 目前 RS-Claw 只能读文件和执行一些安全命令，不会删除系统文件。
   如果担心，可以让它「先告诉我你要做什么，别直接执行」。

**Q: 怎么把它推荐给朋友？**
A: 把你下载的 `rs-claw.exe` 发给朋友，让他们也填自己的 Key 就行。
